 # Channel

## channel01:从一个关闭的channel仍然能够读出数据吗？

从一个有缓冲的 channel 里读数据，当 channel 被关闭，依然能读出有效值。只有当返回的 ok 为 false 时，读出的数据才是无效的。

[channel01](channel/channel01/main.go)

```
received: 19
channel closed,data invalid.
 ```
